# SpeakMySign – Voice and Sign Language Assistant 🚀

## Project Demo
[![SpeakMySign Demo](https://img.youtube.com/vi/bUaKhWh3bbw/0.jpg)](https://youtu.be/bUaKhWh3bbw)

## Overview
SpeakMySign is a web-based application that converts sign language to speech and speech to sign language using AI-based hand gesture detection and speech recognition.

Built to improve accessibility for hearing and speech-impaired users through AI-powered interaction.

## Features
- Sign to Speech using MediaPipe
- Speech to Sign using Speech Recognition
- Learning Module for Sign Language
- Real-time hand gesture detection
- Voice output

## Technologies Used
- HTML
- CSS
- JavaScript
- Bootstrap
- MediaPipe Hands
- Speech Recognition API
- Speech Synthesis API

## How to Run
1. Download the repository
2. Open index.html in browser
3. Allow camera and microphone
4. Use the application
